Class: RequestBuilder

RequestBuilder

Example
```js
nostojs(api => {
  api
    .createRecommendationRequest()
    .setResponseMode("JSON_ORIGINAL")
    .enablePreview()
    .addElements(["frontpage-nosto-1"])
    .load()
    .then(response => {
      console.log(response);
    });
});
```
